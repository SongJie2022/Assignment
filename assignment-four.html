Project Details

More information about the project

Given the problem statement: How might we make mainland students who have just come to Hong Kong, including undergraduate, postgraduate, doctor, and exchange students address the problem of finding like-minded partners to go out with to achieve the goal of enriching after-school life and making like-minded friends?

The scope of the future journey and story mapping includes the following steps:

  1. Get an idea
  2. Learn more about the activity
  3. Make a plan and find partners
  4. Check the plan
  5. Go out and have fun
  6. Comment

Compared to the previous journey, the [Let's play together!] website tries to give users a better user experience in #3. And the website is also helpful in other parts except #5. The website offers the "发现活动" module, which includes the latest and hottest events and other people's shares about activities to give users ideas for activities. Relevant links provided by the website and others' experience allow users to learn more about the event and reduce the problem of bouncing between multiple platforms. The core function of the website is to help users find activity partners, so the website sets "加入活动" and "创建活动" modules. Users can join events organized by other users through "加入活动". They can also organize activities to recruit partners according to their needs. The website also has an internal message box to facilitate communication between users. Finally, the "活动分享" module allows users to share their experiences after participating in an event.

The website makes money mainly through advertising and partnerships. First, the website is aimed at college students studying in Hong Kong. The website adopts the membership system to censor user information. Because the user group is clear, advertising related to students can be put on the page. At the same time, websites can cooperate with event organizers, such as highlighting the event on the home page and the "发现活动" page driving more traffic to the event, to achieve a win-win situation for the website, users and event organizers.

A usability test will also be conducted to validate the monetization logic of engaging the targeted players to subscribe to the community membership. You can find out more about this topic here.
